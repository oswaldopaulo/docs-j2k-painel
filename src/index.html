Painel J2K Sistemas

Clientes

A aba "Clientes" no painel J2K sistemas é destinada ao gerenciamento de informações dos clientes. Nessa aba, os usuários podem acessar e atualizar os dados somente de seus respectivos clientes, fazer a instalação dos programas tanto SNDESK quanto GOCRM, fazer a criação de um banco e a clonagem do sistema para uso.

Para criação de um Cliente:

  • Na aba esquerda em “MENU”.
  • Clique em “CLIENTES”.
  • Na tela de clientes, você verá uma lista de clientes já cadastrados ou estará vazio caso não tenha feito nenhum cadastro ainda.
  • Para adicionar um novo cliente, clique em "+ NOVO CLIENTE".
  • Caso deseja acessar um cliente já cadastrado, basta seleciona-lo.
  • Preencha os campos com as informações do cliente, sendo requerido Nome, Domínio, CPF/CNPJ.


Funções:

Acesse opções em "Escolha seu sistema":
  • INSTALAR MANUALMENTE
    • Programa: selecione o programa(SNDESK ou GOCRM)
    • Url: Insira a url com seu domínio.
    • Após ser adicionado o programa no sistema do cliente, poderá ser visualizado algumas informações da instalação.
  • CRIAR BANCO:
    • Nome do banco é criado com base no domínio.
    • Cria banco e usuário.
  • CLONAR REPOSITÓRIO:
    • Faz a clonagem do sistema no servidor.

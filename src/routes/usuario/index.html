Usuário

A aba "Usuário" no painel J2K sistemas permite a criação e gestão de usuários, podendo criar e gerenciar clientes. Cada usuário só poderá acessar e atualizar os dados dos clientes que ele mesmo criou, garantindo que não tenha acesso aos clientes de outros usuários. Existem dois tipos de usuários:

  • Comum: não tem acesso aos demais usuários.
  • Super: possui acesso a qualquer usuário.

Para criação de um usuário:

  • Na aba esquerda em “MENU”.
  • Clique em "USUÁRIO".
  • Na tela de usuários, você verá uma lista de usuários já cadastrados ou estará vazio caso não tenha feito nenhum cadastro.
  • Para adicionar um novo cliente, clique em "+ NOVO USUÁRIO".
  • Caso deseja acessar um usuário já cadastrado, basta seleciona-lo.
  • Preencha os campos com as informações do usuário, sendo requerido Nome, Tipo e a senha.
